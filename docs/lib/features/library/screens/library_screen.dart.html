library_screen.dart

lib/features/library/screens/library_screen.dart

Overview

Technical Explanation

This file manages the Library Book Lifecycle. It interfaces with LibraryDatabaseHelper for CRUD operations on book records. It includes a built-in notification scheduler that sets alerts for 7, 3, and 1 day(s) before the book's due date using the local notification plugin.

Layman Explanation

This is your Digital Library Shelf. When you borrow a book, you add it here. The app then acts like a helpful librarian who quietly reminds you several times before the book is due, so you never have to pay a late fine again.
