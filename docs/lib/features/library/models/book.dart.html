lib/features/library/models/book.dart

Overview

Technical Explanation

The Library Domain Entity. This model encapsulates library-specific attributes like accession numbers, authors, issue dates, and return deadlines. It includes JSON serialization logic and a copy() method for non-destructive state updates.

Layman Explanation

This is the Digital Library Card for a book. It stores all the information about a book you've borrowed—like the title, the author, and when you're supposed to give it back—so the app can remind you on time.
