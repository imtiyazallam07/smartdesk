library_database_helper.dart

lib/features/library/services/library_database_helper.dart

Overview

Technical Explanation

The Dedicated Database Manager for library records. It handles the books table, managing fields for accession numbers, authors, and return dates. It includes logic to increment versions and handle schema migrations when new features are added.

Layman Explanation

This is the Library Archive. It manages the secret list of books you've saved. It makes sure that all the details—like what book you borrowed and when you have to return it—are written down safely so the app never forgets.
