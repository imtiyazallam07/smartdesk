calendar_sync_service.dart

lib/features/todo/services/calendar_sync_service.dart

Overview

Technical Explanation

Interfaces with the native Device Calendar (Google Calendar/iOS Calendar). It handles Permission requests, finds the user's default writable calendar, and creates/deletes native events matching tasks created within SmartDesk.

Layman Explanation

This amazing feature lets you Sync with your Phone's Calendar. When you add an important reminder in SmartDesk, this service makes sure it also shows up in your main Google or Apple Calendar app, so you see it everywhere.
