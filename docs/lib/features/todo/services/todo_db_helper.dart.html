todo_db_helper.dart

lib/features/todo/services/todo_db_helper.dart

Overview

Technical Explanation

The SQLite Database Singleton for task persisting. It manages the recurring_tasks and one_time_tasks tables. It implements schema versioning (migrations) to handle updates like adding calendar event IDs to existing user data.

Layman Explanation

This is the app's Permanent Filing Cabinet for your tasks. It's responsible for writing down your to-do items into a small file on your phone so they are still there even if you restart your phone or update the app.
