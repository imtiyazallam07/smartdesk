todo_model.dart

lib/features/todo/models/todo_model.dart

Overview

Technical Explanation

Defines the core Domain Entities for the task feature: RecurringTask and OneTimeTask. Includes properties for intervals, priority levels, and notification timestamps. Provides standard logic for converting between JSON (Map) and Dart objects.

Layman Explanation

This file defines the Blueprint for your Tasks. It's a set of rules that decides what information a task should hold (like the name, the deadline, and the priority) so the app can treat all your tasks consistently.
