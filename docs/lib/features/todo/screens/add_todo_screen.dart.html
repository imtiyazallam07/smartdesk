lib/features/todo/screens/add_todo_screen.dart

Overview

Technical Explanation

A Dual-Purpose Input Form for task creation and editing. It dynamically updates its fields based on the selected task type (Recurring vs One-Time). It handles sophisticated validation for intervals, selected days, and deadline dates before persisting to the local DB.

Layman Explanation

This is the Task Creator. Whether you're adding a homework deadline or a workout schedule, you do it here. You can name the task, set the date or frequency, and even decide exactly what time you want the app to remind you about it.
