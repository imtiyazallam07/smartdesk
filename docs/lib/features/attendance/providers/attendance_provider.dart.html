attendance_provider.dart

lib/features/attendance/providers/attendance_provider.dart

Overview

Technical Explanation

A ChangeNotifier that manages the application's attendance state. It implements an In-Memory Cache (HashMap) for fast O(1) data retrieval during UI rebuilds. It handles SQLite synchronization via DatabaseHelper and provides complex data transformations for chart rendering.

Layman Explanation

This is the app's Attendance Ledger. It's the brain that remembers every time you were present or absent. It keeps a copy of your records in "active memory" so the app feels snappy, and it handles all the math required to show you those colorful progress charts.
