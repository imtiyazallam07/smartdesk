subject_provider.dart

lib/features/attendance/providers/subject_provider.dart

Overview

Technical Explanation

A specialized State Provider dedicated to subject list management. It interfaces with the DatabaseHelper and provides methods for loading, adding, and deleting subjects, triggering UI updates via the notifyListeners() pattern.

Layman Explanation

This is the Subject Organizer. It's the part of the app that handles the list of your classes. If you add "Math" to your list, this provider tells every other page in the app about the new subject so it's available everywhere.
