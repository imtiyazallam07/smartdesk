timetable_provider.dart

lib/features/attendance/providers/timetable_provider.dart

Overview

Technical Explanation

A ChangeNotifier class that maintains the Local Schedule State. It maps weekday integers to TimeSlot lists and handles all CRUD (Create, Read, Update, Delete) operations. It ensures that changes are persisted to the database immediately after being updated in memory.

Layman Explanation

This is the Schedule Manager. It's the part of the app that organizes your classes by day and makes sure they aren't lost when you close the app. When you add a class on Tuesday, this "brain" remembers it and makes sure it appears correctly every Tuesday.
