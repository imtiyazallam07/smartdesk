day_attendance_screen.dart

lib/features/attendance/screens/day_attendance_screen.dart

Overview

Technical Explanation

This file implements the Granular Attendance Editor. It maps the selected date to the corresponding weekday's timetable slots. It allows users to mark individual slots as present/absent, calculates a "Partial" status if mixed, and saves the resulting DailyAttendance object to the database.

Layman Explanation

This is where you mark the Class-by-Class Attendance for a specific day. If you only made it to 2 out of 4 classes today, you can check off those two specifically. The app will then accurately reflect your "Partial Attendance" for that day.
