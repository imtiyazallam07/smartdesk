lib/features/attendance/screens/subject_management_screen.dart

Overview

Technical Explanation

Implements Master Data Management for academic subjects. It allows users to define subject entities, which are then used as foreign references in both the Timetable and Attendance tables. It uses Uuid to ensure unique primary keys across multiple device installs.

Layman Explanation

This is where you Add Your Subjects (like "Physics" or "Python"). You need to add them here first so the app knows what subjects to let you pick from when you're setting up your timetable or marking attendance.
