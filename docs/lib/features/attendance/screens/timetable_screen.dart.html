timetable_screen.dart

lib/features/attendance/screens/timetable_screen.dart

Overview

Technical Explanation

This file provides the Tabbed Navigation UI for viewing the weekly schedule. It uses a TabController with a fixed length of 6 (Mon-Sat). It fetches slot data from TimetableProvider and includes modal dialogs for Adding, Editing, and Deleting class slots.

Layman Explanation

This is your Class Schedule page. You can swipe between the days of the week to see what classes you have. It works just like a digital notebook where you can add new classes or delete old ones by tapping on them.
