Overview

Technical Explanation

The Attendance Management Root. It integrates the TableCalendar widget for date selection and uses Consumer widgets to react to attendance updates. It also coordinates data from HolidayService to determine the next relevant working day and update the UI accordingly.

Layman Explanation

This is the Attendance Calendar. It marks the days you attended class (in green) and the days you missed (in red). You can tap any date to see exactly what classes you had that day, and it even tells you when your next class is scheduled at the top.
