lib/features/attendance/screens/timetable_setup_screen.dart

Overview

Technical Explanation

The Timetable Configuration Interface. It uses ExpansionTile widgets for each day and provides a complex multi-step dialog for defining class slots (Subject Selection + Time Picking). It facilitates direct communication with TimetableProvider to sync data.

Layman Explanation

This is the Timetable Builder. It's a special setup page where you can build your weekly class schedule from scratch. You can tap on a day of the week and add classes one by one, picking the subject and the time for each class.
