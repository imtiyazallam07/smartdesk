subject.dart

lib/features/attendance/models/subject.dart

Overview

Technical Explanation

A Plain Old Dart Object (PODO) representing an academic subject. It includes a unique id and a descriptive name. It provides standard serialization methods to interface with the SQLite database layer.

Layman Explanation

This is a Digital Nametag for a subject. It's just a simple container that stores the name of a class (like "Math") and a secret ID number so the computer can keep track of it reliably.
