timetable.dart

lib/features/attendance/models/timetable.dart

Overview

Technical Explanation

Defines the TimeSlot and DaySchedule data structures. TimeSlot encapsulates start and end times as integer pairs (hour/minute) and includes a timeString getter that performs 12-hour AM/PM formatting for display.

Layman Explanation

This file defines what a Class Entry looks like. It stores the name, start time, and end time of a class. It also includes the logic to turn numbers like "14:30" into friendly text like "2:30 PM".
