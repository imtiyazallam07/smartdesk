holiday_service.dart

lib/features/attendance/services/holiday_service.dart

Overview

Technical Explanation

This service provides Business Logic for Schedule Validations. It parses holiday lists from cache and offers utility methods like getNextWorkingDay() which intelligently skips Sundays and holidays to find the next valid academic day for planning.

Layman Explanation

This is the app's Holiday Specialist. It's the brain that knows when you have a day off. It's used to make sure the app doesn't set alarms on Sundays or during festivals, and it helps the dashboard show you exactly when your next real class day is.
