database_helper.dart

lib/features/attendance/services/database_helper.dart

Overview

Technical Explanation

The Centralized Data Persistence Engine for attendance and timetables. It manages the attendance_tracker.db including tables for subjects, slots, and daily_attendance. It ensures transactional integrity and uses conflict resolution strategies for data reliability.

Layman Explanation

This is the app's Safe-Deposit Box. Whenever you add a class or mark your attendance, this service writes it down in a permanent, secure file on your phone. It's the reason why your data is still there whenever you reopen the app.
