lib/features/attendance/attendance_home.dart

Overview

Technical Explanation

A Barrel File (Export Layer) designed for architectural cleanliness. It re-exports the main Attendance screen, allowing other modules to import the entire attendance feature through a single, clean endpoint without worrying about internal folder structures.

Layman Explanation

Think of this like a Feature Shortcut. It doesn't contain the actual code for the screens, but it acts as a central door that helps other parts of the app find the Attendance Tracker easily.
