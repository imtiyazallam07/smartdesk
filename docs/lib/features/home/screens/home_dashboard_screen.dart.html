Overview

Technical Explanation

The HomeDashboardScreen is a sophisticated data aggregator. It consumes state from AttendanceProvider, TimetableProvider, and TodoDatabaseHelper to provide a unified summary. It calculates "Critical Subjects" (where attendance is < 80%) and renders dynamic charts using the fl_chart library.

Layman Explanation

Think of this as your Personal Assistant's Briefing Room. As soon as you open the app, it looks at your homework, your library books, and your attendance percentages. If you're missing too many classes or have a task due soon, it will flag it here in big clear cards.
