lib/features/home/screens/smart_desk_screen.dart

Overview

Technical Explanation

This file implements the Primary Application Scaffold. It manages the BottomNavigationBar and coordinates navigation between the major feature modules: Dashboard, Curriculum, and Calendar. It also includes an automated _checkForUpdates method that performs an HTTP GET request to check for new APK versions on GitHub.

Layman Explanation

This is the Home Base of the app. It's the "frame" that holds the bottom menu icons. Whether you're checking your classes or looking at the calendar, this screen stays in the background to help you switch views. It also keeps an eye out for updates to make sure you're using the latest version.
