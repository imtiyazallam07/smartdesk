Overview

Technical Explanation

The Application Meta Viewer. It displays high-level information including the semantic version number and branding assets. It implements url_launcher to provide external intent-launching for developer social profiles and repositories.

Layman Explanation

This is the Info Page. It's like the credits at the end of a movie. It shows you what version of the app you're using and provides links to the developer's profiles if you want to give feedback or follow the project.
