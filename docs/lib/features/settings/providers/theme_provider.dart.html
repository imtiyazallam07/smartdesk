theme_provider.dart

lib/features/settings/providers/theme_provider.dart

Overview

Technical Explanation

A ChangeNotifier that serves as the Theme Management Service. It encapsulates the app's ThemeMode state and uses SharedPreferences to provide persistent storage. It ensures that theme transitions (System, Light, Dark) are broadcasted across the entire widget tree.

Layman Explanation

This is the app's Lighting Designer. It remembers if you prefer a Dark mode or Light mode. It also makes sure that when you change the theme, every single page in the app switches instantly to match your preference.
