Overview

Technical Explanation

The Global Preference Interface. It allows users to toggle themes and set their "Joining Year". It includes original logic to calculate the student's current academic year (e.g., "3rd Year") by comparing the joining date with the current system date.

Layman Explanation

This is the Control Panel for the app. Here you can change the look of the app or tell it when you joined college. Based on your joining year, the app smartly figures out if you are a Freshman, Sophomore, or Senior so it can show you the right information.
