lib/features/calendar/screens/calendar_screen.dart

Overview

Technical Explanation

Implements a State-Driven Holiday Viewer. It fetches JSON data from a GitHub source and implements an offline-first caching strategy via SharedPreferences. It separates data into General Holidays and Academic Calendars (per-year schedules).

Layman Explanation

This is the Holiday List. It shows you all the official days off (like festivals). Even if you have no internet, it remembers the list from the last time you checked. It also lets you jump to your specific year's academic calendar (e.g., "2nd Year Schedule").
