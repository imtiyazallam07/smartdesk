curriculum_screen.dart

lib/features/curriculum/screens/curriculum_screen.dart

Overview

Technical Explanation

A Remote Data-Driven Viewer. It fetches a curriculum JSON from a configured endpoint and employs an Offline-Cache Layer using SharedPreferences. It renders a hierarchical list of subjects, modules, and topics based on the structure of the JSON payload.

Layman Explanation

This is your Syllabus Guide. It downloads the latest list of topics you need to study from the internet and saves a copy on your phone. This way, you can check what's in your curriculum even when you're in a classroom with no Wi-Fi.
