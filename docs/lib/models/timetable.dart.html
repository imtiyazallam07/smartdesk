timetable.dart

lib/models/timetable.dart

Overview

Technical Explanation

The Shared Timetable Definition. It provides the core structures (TimeSlot, DaySchedule, WeeklySchedule) used in the root of the project to define how academic schedules are organized and displayed.

Layman Explanation

This file is the Blueprint for your week. It defines how a single class, a whole day, and even a full week of classes should be structured so the app can show you your timetable correctly.
