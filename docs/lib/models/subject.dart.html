Overview

Technical Explanation

A Global Domain Model for academic subjects. It provides the structured definition (ID and Name) used shared across multiple features. This version acts as the root model found in the project's base directory.

Layman Explanation

This is a Shared Tool that lets the app recognize what a "Subject" is. It helps the computer keep track of names like "Math" or "Code" efficiently across the whole application.
