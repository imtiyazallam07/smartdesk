web_view_screen.dart

lib/shared/widgets/web_view_screen.dart

Overview

Technical Explanation

A robust Embedded Browser Implementation. It wraps webview_flutter and provides a standard UI with a dual-purpose AppBar that includes loading progress indicators and navigation controls (back, forward, reload) for seamless web browsing.

Layman Explanation

This is a Built-in Web Browser. It lets you open websites (like college notices or syllabus links) right inside the app, so you don't have to constantly jump back and forth between SmartDesk and your phone's browser.
