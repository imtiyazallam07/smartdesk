lib/widgets/dashboard_card_widget.dart

Overview

Technical Explanation

A Reusable Atomic UI Component. It standardizes the visual structure of dashboard modules, including a title header, an optional icon, a "View All" interaction callback, and a flexible content slot that supports any nested widget tree.

Layman Explanation

This is the Blueprint for Dashboard Cards. It's a template that ensures every card on your home screen (like "Tasks" or "Classes") looks consistent, with the same rounded corners and "View All" button in the same place.
